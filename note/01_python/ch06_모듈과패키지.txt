6장 모듈과 패키지

모듈/패키지 모듈안의 함수를 가져오는 방법

import A             - A모듈을 가져옴
import A as B        - A모듈을 B이름으로 가져옴
from A import B      - A모듈안의 B함수(클래스|변수)를 가져옴
from A import B as C - A모듈안의 B함수를 C이름으로 가져옴

from A import B      - A패키지안의 B모듈을 가져옴
from A.B import C    - A패키지안의 B모듈안의 C함수 가져옴
from A import B as C - A패키지안의 B모듈을 C이름으로 가져옴
from A.B improt C as D - A패키지안의 B모듈안의 C를 D이름으로 가져옴

파이썬이 폴더를 패키지에 포함하도록 처리하기 위해 필요한 파일
__init__,py

패키지에 있는 서브 모듈을 모두 가져와 이용할때
improt* 

패키지 작성자가 패키지의 명시적 색인을 제공하기 위해 사용하는 속성의 이름
__all__

아나콘다 프롬프트에서
- 패키지 설치 : pip install 패키지명[==버전] / conda install 패키지명[==버전] / uv pip install 패키지면[==버전]
- 패지키 확인 : pip show 패키지명           / conda list 패키지명
- 패키지 삭제 : pip uninstall 패키지명      / conda remove 패키지명
