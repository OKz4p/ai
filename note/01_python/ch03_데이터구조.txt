3장 데이터구조 : 여러 데이터를 한 변수에 담는게 목적
1절. 리스트[] : 타입에 상관없이 0개 이상의 데이터들을 담을 수 있다.
	1.1 fltmxm todtjd : [], list(range(from, to, step)) == 나열 가능한 자료
	1.2 기본 정보 조회 
		- min(lst), max(lst), sum(lst), len(lst)
		- sorted(lst, reverse=True) : 정렬된 lst를 반환
		- lst.sort() : lst가 정렬된 값으로 변경 # sorted와 sort 둘다 reverse=True를 넣으면 내림차순으로 된다
	1.3 요소 추가
		- lst.append(value), extend([values]), insert(idx, value)
	1.4 인덱싱과 인덱스 찾기
		- lst[idx] # lst == 리스트 변수, idx == 인덱스
		- lst.index(찾을 데이터) : 없는 데이터를 찾는 경우 error
		- 찾을데이터 in lst : 찾을 데이터가 있는지 여부 반환
		- lst.count(찾을데이터) : 찾을 데이터가 몇개인지 숫자 반환
	1.5 원하는 요소 추출
		- 슬라이싱
		- 리스트컴프리헨션(원하는 데이터 추출)
	1.6 요소 수정 : 인덱싱이나 슬라이싱을 이용
	1.7 요소 삭제 : pop(), pop(idx), del lst[idx], remove(데이터)
	1.8 복제(깊은복사) : 복제할 변수 = 원본데이터.copy()

3절. 딕셔너리
	3.1 딕셔너리 만들기
		- 중괄호{} 를 이용해 생성, key 와 value(값) 으로 구성된 자료구조 ex. dic = {"name" : "홍길동"}
		- key는 유일한 값(중복X) / 값은 중복 허용
		- 인덱스를 이용한 참조 지원X
	3.2 딕셔너리 item 수정, 추가, 제거
		- 한 item씩 추가
			: 변수["추가할key"] = "추가할값"
		- 여러 item을 한번에 추가
			: 변수.update({"추가할 key1" : "추가할 값1", "추가할 key2" : "추가할 값2"})
		- 한 item씩 수정
			: 변수["생성한 key"] = "수정할 값"
		- 여러 item 한번에 수정
			: 변수.update(생성한 key1 = "수정할 값1", 생성한 key2 = "수정할 값2"
		- item 제거방법 1
			: del 변수["삭제할key"] - 없는 key일 경우 에러
		- item 제거방법 2
			: 변수.pop("삭제할key") - 해당 key값을 반환하고 제거(없는 key일 경우 에러)
